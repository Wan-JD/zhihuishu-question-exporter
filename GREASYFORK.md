# 智慧树题库导出助手

用于把智慧树答题记录和解析页中的题目导出为 Markdown 复习文档。适合课后整理、错题复盘和期末复习。

## 主要功能

- 在解析页收录题干、选项、参考答案、解析。
- 默认导出干净文本，不导出图片，不附带页面原始文本。
- 自动清理 `font-size`、`mso-` 等 Word/CSS 样式残留。
- 自动去掉“回答正确/回答错误”等提示，避免污染选项。
- 支持同一题多次出现时去重合并。
- 支持按当前章节导出，也支持汇总已收录题库后导出全课 Markdown。

## 使用方法

1. 安装脚本后进入智慧树课程页或题目解析页。
2. 在解析页点击右下角面板中的“收录本页”。
3. 回到课程页后可点击“保存全课”或“下载全课”导出。
4. 如果需要批量跨章节收录，可在课程页使用“开始历史收录”。

## 注意事项

- 脚本依赖你账号已有的答题记录和解析页内容，不会凭空生成未出现过的隐藏题。
- 数据保存在浏览器本地 `localStorage`，更换浏览器或清理站点数据后需要重新收录。
- 脚本只用于整理个人学习记录，请遵守课程平台和学校相关规定。

## 隐私说明

脚本不上传题目、账号信息或导出内容；所有收录数据均保存在本机浏览器。
